# 🔥 Fire Fighting Robot using Arduino (Work in Progress)

## 📌 Overview
This project is an autonomous fire fighting robot built using Arduino.
The robot is designed to detect fire using sensors and automatically activate a water pump to extinguish it.

⚠️ This project is currently under development.

---

## 🚀 Planned Features
- Fire detection using flame sensor
- Automatic movement using 4 DC motors
- Relay controlled water pump
- Multi-sensor detection
- Arduino based control

---

## 🛠️ Components Used
- Arduino Uno
- Flame Sensor
- Temperature Sensor
- Smoke Sensor
- Motor Driver
- DC Motors
- Relay Module
- Water Pump
- Battery
- Robot Chassis
